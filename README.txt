Программа для парсинга файлов из chemcad и их записи в базу данных
Входные данные:
1. XML файл с различными атрибутами - необходимо запарсить и внести в таблицы базы данных
2. TXT файл с разными таблицами - необходимо выбрать нужное и внести в базу
3. DXF файл - необходимо конвертировать в SVG формат
Выходные данные:
1. Данные с txt отчета в формате json (чтобы могла подхватить MasterScada)
СУБД должна работать по сути в формате библиотеки, предоставляя пользователю возможность создания записей в этой библиотеке. В рамках создания записи пользователю должна предоставляться возможность размещения (посредством обзора и выбора) файла Chemcad (формата *.cc6), файла DXF, файла XML и файла форматов RTF или TXT. После включения файла DXF в библиотеку в интерфейсе должно быть предусмотрено отображение изображения DXF в формате PNG, например. После включения файла XML в библиотеку в интерфейсе должно быть предусмотрено отображение параметров, присутствующих в XML, и их значений. Вся созданная пользователем запись должна сохраняться, то есть должна предоставляться возможность открытия пользователем сохраненной записи с привязкой ко всем включенным файлам. 

В библиотеке должен быть организован поиск записей в библиотеке по типу моделей и входным данным. Соответственно должна быть предусмотрена типизация (назначение типа) создаваемой записи и поиск по входным данным среди записей одного типа. Проще говоря, записи одного типа - это, например, модели одинаковой структуры с одинаковыми наборами параметров, но сами параметры в этих моделях могут отличаться численно.
